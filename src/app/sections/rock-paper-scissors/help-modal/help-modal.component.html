Rock, Paper, and Scissors model instruction

This application is an implementation of model that classifies hand gestures of Rock, Paper, and Scissors captured by a webcam.

All data captured by the webcam IS NOT SENT to the server and DOES NOT LEAVE your computer. All calculations take place in your browser, so usage is completely CONFIDENTIAL.

To use the application you should follow these steps:

  1. Make sure the camera is working and you can see the image on the screen
  2. Make at least 30 pictures of each gesture by pressing the corresponding button
  3. Press "Train Network" button to train model
  4. Once training is complete, click 'Start Predicting' to see predictions, and 'Stop Predicting' to end

Note: It can be a bit tricky to train a model well. So, don't get frustrated if you don't get it right the first time.
